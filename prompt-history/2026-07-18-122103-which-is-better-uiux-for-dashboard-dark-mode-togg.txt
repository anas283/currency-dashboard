❯ which is better uiux for dashboard dark mode toggle, emojis / icons / plain   
  text?                                                                         

⏺ Icons win. Reason:

  - Plain text ("Dark Mode" / "Light Mode") — clear but bulky, bad for tight nav
  bars.
  - Emojis (🌙☀️) — render inconsistent across OS/fonts, look unprofessional in
  polished UI, hard color-match to theme.
  - Icons (sun/moon SVG, e.g. lucide/heroicons) — scale clean, inherit
  currentColor for theme, universally recognized pattern (GitHub, Twitter/X,
  every major dashboard use this).

  Best practice: icon-only toggle button, sun↔moon swap on click, add
  aria-label="Toggle dark mode" for accessibility. Optional: pair icon + tooltip
  if space allow.
